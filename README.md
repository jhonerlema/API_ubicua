# Sistema de facturacion

## Crear proyecto

> Garantice que tenga `node` y `npm` instalados.

Ejecute:

```sh
    mkdir project_name
    cd project_name
    npm inti -y # crear el proyecto node 
    # instalar primeras dependecias 
    npm install express
```

* La carpeta ``node_modules`` no deberia al repo
Para esto ulizaremos **.GITINGNORE**.
